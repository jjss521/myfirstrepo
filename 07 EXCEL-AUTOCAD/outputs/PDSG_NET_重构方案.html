五、核心重写：AutoCAD 绘图层

这是唯一需要完全重写的模块。Python 通过 COM 跨进程调用 AutoCAD，C# 通过托管 API 直接在进程内操作数据库。

Python COM → C# 托管 API 映射

Python COM 操作C# 托管 API说明
comtypes.client.GetActiveObject(progid)Autodesk.AutoCAD.ApplicationServices.Application获取AutoCAD实例
app.ActiveDocumentApplication.DocumentManager.MdiActiveDocument当前文档
doc.ModelSpacedoc.Database.CurrentSpaceId模型空间
modelSpace.AddLine(start, end)new AcDbLine(start, end) + btr.AppendEntity()画线
modelSpace.InsertBlock(pt, name, ...)new AcDbBlockReference(pt, name) + btr.AppendEntity()插入图块
ent.SetAttribute(tag, value)ref.AttributeCollection + new AcDbAttribute()写属性
doc.SaveAs(path)doc.Database.SaveAs(path, DwgVersion)保存DWG
time.sleep(2) + _com_retryTransaction 自动管理，无需sleep消除所有等待
_clean_model_space 20轮循环Database.Erase() 一次完成消除索引重排问题
SendCommand("_.LINE ...")直接调用 AcDbLine 构造函数消除命令注入风险

CadDrawer.cs 核心骨架

using Autodesk.AutoCAD.ApplicationServices;
using Autodesk.AutoCAD.DatabaseServices;
using Autodesk.AutoCAD.Geometry;

public class CadDrawer : IDisposable
{
    private Document _doc;
    private Database _db;
    private bool _disposed;

    public void Connect()
    {
        _doc = Application.DocumentManager.MdiActiveDocument;
        _db = _doc.Database;
    }

    public void Draw(LayoutResult layout)
    {
        using (Transaction tr = _db.TransactionManager.StartTransaction())
        {
            BlockTable bt = tr.GetObject(_db.BlockTableId, OpenMode.ForRead) as BlockTable;
            BlockTableRecord btr = tr.GetObject(
                _db.CurrentSpaceId, OpenMode.ForWrite) as BlockTableRecord;

            // 插入所有回路图块
            foreach (var p in layout.Placements)
            {
                InsertBlock(btr, bt, tr, p);
            }

            // 绘制母线
            DrawBusLine(btr, tr, layout.BusLine);

            tr.Commit();  // 原子提交，无索引重排
        }
    }

    private void InsertBlock(BlockTableRecord btr, BlockTable bt,
                             Transaction tr, Placement p)
    {
        if (!bt.Has(p.BlockName)) return;

        var blkRef = new BlockReference(
            new Point3d(p.X, p.Y, 0), bt[p.BlockName]);
        btr.AppendEntity(blkRef);
        tr.AddNewlyCreatedDBObject(blkRef, true);

        // 写入属性
        foreach (var attr in p.Attributes)
        {
            var attDef = GetAttributeDefinition(bt, tr, p.BlockName, attr.Key);
            if (attDef == null) continue;

            var attRef = new AttributeReference();
            attRef.SetAttributeFromBlock(attDef, blkRef.BlockTransform);
            attRef.TextString = attr.Value;
            blkRef.AttributeCollection.Append(attRef);
            tr.AddNewlyCreatedDBObject(attRef, true);
        }
    }

    public void Dispose()
    {
        if (!_disposed) { _disposed = true; }
    }
}

消除的 6 类 COM 卡死问题

COM 问题Python 症状.NET 解决方案
RPC_E_CALL_REJECTED_com_retry 重试3次 + sleep无跨进程调用，彻底消除
time.sleep 硬编码18处 sleep(1~2秒)Transaction 原子操作，无需等待
STA 线程地狱_ensure_sta_com 复杂初始化托管API自动处理线程
实体删除索引重排_clean_model_space 20轮循环Database.Erase() 一次完成
文件锁/保存失败save_as 备用路径兜底进程内操作，无文件锁
SendCommand 注入字符串拼接命令直接调用API，无命令注入
