## 模块详细设计

### main.py -- 流程编排

```
main(argv):
  1. parse_args -> Args(excel_path, config_path?, output_path?)
  2. cfg = load_config(config_path or "config.yaml")
  3. records, errors = excel_reader.read_and_validate(excel_path, cfg)
  4. if not records: print errors; exit
  5. mapped = block_mapper.map(records, cfg)
  6. layout = layout_engine.compute(mapped, cfg)
  7. cad = CadDrawer()
     cad.connect(cfg.autocad)
     cad.load_blocks(cfg.block_library)
     cad.draw(layout, cfg)
     if cfg.title_block.enabled:
       cad.insert_title_block(cfg.title_block)
     cad.save_as(cfg.output.dwg_path)
  8. reporter.generate(errors, mapped, cfg)
  9. print_summary()
```

### excel_reader.py -- Excel读取器

核心函数签名:
```python
def read_and_validate(path: str, cfg: ExcelConfig) -> (List[CircuitRecord], List[ErrorRecord])
```

处理流程:
1. openpyxl.load_workbook()打开
2. 定位Sheet -> 读header行 -> 建立列号到文本的映射
3. 与column_aliases做三级模糊匹配: 精确 -> 别名 -> 包含
4. 必填列匹配率<70%则退出并报错(列出已识别的列)
5. 逐行读取 -> 类型转换 -> 逐校验规则检查
6. 返回(有效记录列表, 错误记录列表)

### block_mapper.py -- 图块映射器

核心函数签名:
```python
def map(circuits: List[CircuitRecord], cfg: BlockMappingConfig) -> List[CircuitWithBlock]
```

匹配算法:
1. 遍历block_mapping.rules列表(有序,优先级从前到后)
2. 对每条CircuitRecord,取(load_type, poles)组合
3. 与规则的match条件比较,首次命中即返回对应block名
4. 无匹配时输出警告并使用default_block
5. 验证匹配到的block名在block_catalog.yaml中确实存在

### attribute_builder.py -- 属性组装器

核心函数签名:
```python
def build(circuit: CircuitRecord, block_def: BlockDefinition) -> Dict[str, str]
```

转换规则:
- 遍历block_def.attributes(该图块拥有的属性Tag列表)
- 从CircuitRecord中取对应字段值
- 数值转为字符串并附加单位(如35.0 -> "35.0kW")
- 空值对应属性设为空字符串""
- 变频器字段: 仅当负荷类型为变频时才填写VFD相关属性

### layout_engine.py -- 布局引擎

核心函数签名:
```python
def compute(circuits: List[CircuitWithBlock], cfg: LayoutConfig) -> LayoutResult

@dataclass
class LayoutResult:
    placements: List[Placement]
    bus_line: BusLine
    group_labels: List[GroupLabel]
    paper_size: PaperSize

@dataclass
class Placement:
    block_name: str
    x: float
    y: float
    attributes: Dict[str, str]
    circuit_id: str

@dataclass
class BusLine:
    x: float
    y_start: float
    y_end: float
```

核心算法见布局伪码。GroupLabel在分组间插入, Placements按Y坐标降序排列。

### cad_drawer.py -- AutoCAD绘图器

核心类与接口:
```python
class CadDrawer:
    def connect(self, acad_cfg: AcadConfig) -> bool
    def load_blocks(self, lib_cfg: BlockLibraryConfig) -> None
    def draw(self, layout: LayoutResult, cfg: LayoutConfig) -> None
    def draw_bus(self, bus_line: BusLine) -> None
    def place_block(self, placement: Placement) -> None
    def insert_title_block(self, cfg: TitleBlockConfig) -> None
    def save_as(self, path: str) -> None
```

关键AutoCAD COM调用链:
```python
# 连接
acad = Autocad(create_if_not_exists=False)

# 导入图块库(将外部DWG中的Block定义导入当前文档)
acad.doc.SendCommand(f'(command "-INSERT" "{block_library_path}" ^C^C)')

# 插入图块引用
block_ref = acad.model.InsertBlock(
    APoint(x, y, 0),    # 插入点
    block_name,          # 图块名
    1.0, 1.0, 1.0,      # X/Y/Z比例
    0.0                  # 旋转角度
)

# 填写属性
for attr in block_ref.GetAttributes():
    if attr.TagString in attrs_dict:
        attr.TextString = attrs_dict[attr.TagString]

# 画母线(粗实线)
line = acad.model.AddLine(APoint(x1,y1,0), APoint(x2,y2,0))
line.Lineweight = 50  # 0.5mm = 50(单位: 0.01mm)
```
