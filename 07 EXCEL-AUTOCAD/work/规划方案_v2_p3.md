# Part 2: Excel模板与布局算法

## Excel数据模板

### 标准表结构

工作簿要求: Sheet名称为回路清单(可配置), 第1行列标题, 第2行起回路数据, 数据区无合并单元格。

标准列定义:

| 列 | 标准列名 | 别名 | 类型 | 必填 | 说明 |
|---|---|---|---|---|
| A | 回路编号 | 编号 | str | 是 | 全局唯一 |
| B | 回路名称 | 名称,用途 | str | 是 | |
| C | 负荷类型 | 类型 | enum | 是 | 动力/照明/变频等 |
| D | 额定功率 | 功率(kW),Pe | float | 是 | 单位kW |
| E | 额定电流 | 电流(A),Ijs | float | 是 | 单位A |
| F | 断路器型号 | 开关型号 | str | 是 | |
| G | 断路器极数 | 极数,Poles | int | 是 | 1/2/3/4 |
| H | 脱扣器电流 | Ir,整定电流 | float | 是 | 单位A |
| I | CT变比 | 互感器变比,CT | str | 是 | 如100/5A |
| J | 电缆型号 | 电缆 | str | 是 | |
| K | 电缆规格 | 截面,规格 | str | 是 | 如4x25+1x16 |
| L | 变频器型号 | 变频器 | str | 否 | |
| M | 变频器功率 | VFD功率 | float | 否 | 单位kW |
| N | 备注 | Remark | str | 否 | |

### 负荷类型枚举

| 枚举值 | 含义 | 需变频器? |
|---|---|---|
| 动力 | 电动机/动力配电回路 | 否 |
| 照明 | 照明配电回路 | 否 |
| 变频 | 含变频器驱动回路 | 是 |
| 空调 | 空调/暖通回路 | 否 |
| 插座 | 插座配电回路 | 否 |
| 备用 | 预留备用回路 | 否 |
| 电容补偿 | 无功补偿回路 | 否 |

### 数据校验规则

| 校验项 | 规则 | 违反处理 |
|---|---|---|
| 回路编号唯一性 | 不得重复 | 标记跳过后续相同编号 |
| 必填非空 | 必填列不可为空 | 标记跳过该行 |
| 负荷类型枚举 | 必须在枚举表内 | 标记跳过该行 |
| 变频器一致性 | 类型=变频时VFD型号必填 | 标记跳过该行 |
| 数值范围 | 功率>0,电流>0,极数=1-4 | 标记跳过该行 |
| 列名匹配率 | 至少70%必填列可匹配 | 否则退出报错 |

### 容错设计

- 未知列: 忽略,仅debug日志
- 可选列空值: 正常处理,图块属性留空
- 大小写不敏感匹配
- 数值自动转换: 100A -> 100.0
- 合并单元格: openpyxl读取实际值填充

## 布局算法

### 坐标系

(0,0)=图纸左下角, X轴向右, Y轴向上。母线为垂直线位于图纸左侧(X=bus_x)。各回路图块沿母线从上到下排列(Y递减)。

### 布局参数

| 参数 | 默认值(mm) | 说明 |
|---|---|---|
| bus_x | 100 | 母线X坐标 |
| block_offset_x | 15 | 图块距母线右侧偏移 |
| start_y | 380 | 第一个回路距顶边 |
| vertical_spacing | 70 | 回路垂直间距 |
| margin_top | 15 | 上边距 |
| margin_bottom | 30 | 下边距 |

### 自动幅面选择

总高度 = margin_top + 回路数*vertical_spacing + margin_bottom。
总高度<=420: A2(594x420)  420< <=594: A2横向  594< <=841: A1  841< <=1189: A0

### 排序规则

1. 按负荷类型分组: 动力->变频->空调->照明->插座->备用->电容补偿
2. 组内按回路编号自然排序
3. 可选: 分组间插入分隔标签

### 母线绘制

从最上方回路母排接入点画至最下方, 线宽0.5mm粗实线。各回路通过内部母排接入点自动对齐。

### 布局伪码

```
compute_layout(circuits, config):
    groups = group_and_sort(circuits, config.sort)
    total = sum(len(g) for g in groups)
    paper = select_paper(total * spacing + margins)
    y = paper.height - margin_top
    placements = []
    for group in groups:
        if group_separator_enabled:
            placements.append(GroupLabel(group.name, X, y))
            y -= spacing
        for c in group.circuits:
            placements.append(Placement(c.block_name, bus_x+offset, y, attrs))
            y -= spacing
    bus = Line((bus_x, first_y), (bus_x, last_y))
    return LayoutResult(placements, bus, paper)
```
