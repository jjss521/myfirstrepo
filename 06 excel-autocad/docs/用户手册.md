## 1. 软件简介

### 1.1 PDSG 是什么

PDSG（Power Distribution Single-line Generator）是一款面向电气设计工程师的自动化工具，用于从 Excel 回路清单自动生成 AutoCAD 配电系统单线图。

典型工作场景：你有一份描述配电柜各回路参数的 Excel 表格（含断路器型号、电缆规格、负荷值、CT 变比等），需要在 AutoCAD 中逐个回路绘制系统图。PDSG 可以一键完成这项工作——读取 Excel、匹配图块、计算布局、填写属性、绘制母线，最终输出完整的 DWG 文件。

### 1.2 核心能力

- 自动读取 Excel 回路清单，支持中英文列名模糊匹配
- 自动检测 Excel 格式：标准格式（一行一条回路）和转置格式（参数在行、回路在列）
- 按"负荷类型 + 极数"自动匹配回路图块
- 自动计算布局坐标：母线垂直排列、回路等距分布、自动选择图纸幅面
- 图块属性自动填写（回路编号、断路器型号、电缆规格等）
- 完整的错误处理：单回路出错跳过继续，全部错误写入 HTML 报告
- 双模式运行：GUI 图形界面 / CLI 命令行

### 1.3 支持环境

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10/11 |
| Python | 3.10 及以上 |
| AutoCAD | 2020 / 2022 |
| 电压等级 | 400V 低压配电系统 |
| 回路数量 | 数十个至 200 个 |

## 2. 安装与环境准备

### 2.1 安装 Python

1. 从 https://www.python.org/downloads/ 下载 Python 3.10+ 安装包
2. 安装时务必勾选 **"Add Python to PATH"**
3. 安装完成后在命令行验证：

```
python --version
```

应显示 `Python 3.10.x` 或更高版本。

### 2.2 安装依赖库

打开命令提示符（cmd），进入项目目录：

```
cd C:\Users\你的用户名\Documents\Codex\2026-06-20\excel-autocad
```

安装 Python 依赖：

```
pip install -r requirements.txt
```

依赖清单：

| 库名 | 用途 |
|------|------|
| openpyxl | 读取 Excel (.xlsx) 文件 |
| pandas | 数据清洗和批量映射 |
| pyautocad | AutoCAD COM 互操作 |
| comtypes | COM 底层通信 |
| PyYAML | 配置文件解析 |
| Jinja2 | HTML 报告生成 |
| pytest | 测试框架（开发用） |

> **注意**：tkinter 是 Python 内置库，无需额外安装。

### 2.3 确认 AutoCAD 已安装

PDSG 通过 COM 接口驱动 AutoCAD，需要：

1. AutoCAD 2020 或 2022 已安装在本机
2. 生成 DWG 时 AutoCAD 需要处于运行状态（PDSG 会自动连接）
3. 仅做 Dry-Run 校验预览时无需启动 AutoCAD

### 2.4 项目目录结构

```

  main.py                  CLI 入口（命令行模式）

  src/                     核心源代码

    block_catalog.yaml     图块目录定义

  docs/                    文档（本手册）
```

## 3. 快速开始（5分钟上手）

### 步骤一：准备 Excel 回路清单

按照 [第6节](#6-excel-回路清单填写规范) 的规范准备 Excel 文件。Sheet 名称设为"回路清单"，第 1 行列标题，第 2 行起填数据。

快速验证：项目自带测试文件 `tests/fixtures/sample_valid.xlsx`，可直接使用。

### 步骤二：准备图块库

在 `blocks/block_library.dwg` 中按 [第8节](#8-图块库创建规范) 创建回路图块。每个图块命名遵循 `LOOP_<类型>_<极数>_<序号>` 规范，并包含标准属性 Tag。

快速验证：项目自带 `blocks/block_catalog.yaml`，描述了 12 种预定义图块。

### 步骤三：启动 GUI

```
python gui.py
```

或：

```
python main.py --gui
```

### 步骤四：选择文件并执行

1. 点击"浏览..."选择 Excel 文件
2. 在"配置"标签页检查/调整参数
3. 点击"数据预览"标签页，点"加载 Excel 数据"查看
4. 点击"校验预览 (Dry-Run)"先验证数据是否正确
5. 确认无误后点击"生成 DWG"（需 AutoCAD 已启动）
6. 点击"打开报告"查看 HTML 报告

### 步骤五：查看结果

- 输出 DWG：`output/配电系统图.dwg`（或自定义路径）
- HTML 报告：`output/report.html`
- 运行日志：`logs/app.log`

## 4. GUI 图形界面详解

### 4.1 界面总览

GUI 窗口分为四个区域：

- **顶部文件栏**：选择 Excel 文件和输出路径
- **标签页**：配置 / 数据预览 / 执行
- **底部状态栏**：当前状态和耗时

### 4.2 顶部文件栏

| 控件 | 说明 |
|------|------|
| Excel 路径 | 选择回路清单 .xlsx 文件，点击"浏览..."打开文件选择器 |
| 输出路径 | 指定生成的 DWG 文件保存位置，点击"浏览..."选择 |

快捷键：`Ctrl+O` 打开 Excel 文件选择。

### 4.3 配置标签页

配置标签页包含五个区块，从上到下依次为：

#### Excel 读取

| 选项 | 说明 | 默认值 |
|------|------|--------|
| 自动检测 Excel 格式 | 自动判断 Excel 是标准格式（一行一条回路）还是转置格式（A列为参数名，B/C/...列为各回路数据） | 勾选 |
| Sheet 名称 | 要读取的 Excel 工作表名称。"低压配电系统"与"回路清单"互为自动备选 | 低压配电系统 |
| 默认断路器型号 | 当 Excel 中未提供断路器型号列时，统一使用此型号填入 | NSX100N |

> 转置格式的 Excel 通常由设计院导出，参数名称在第一列（A列），每一列对应一个回路。程序会自动识别并正确解析。

#### AutoCAD 连接

| 选项 | 说明 | 默认值 |
|------|------|--------|
| AutoCAD 可见 | 连接后是否显示 AutoCAD 窗口 | 勾选 |
| ProgID 列表 | AutoCAD COM 标识符，多个用逗号分隔。程序按顺序尝试连接 | AutoCAD.Application.24.1, AutoCAD.Application.23.1 |

> 如果你使用的是 AutoCAD 2020，可将 24.1 的条目删除或调换顺序。

#### 图块库文件

| 选项 | 说明 | 默认值 |
|------|------|--------|
| 图块库 DWG | 包含所有回路图块的 DWG 文件路径 | ./blocks/block_library.dwg |
| 图块目录 YAML | 描述图块名称、适用类型和属性列表的 YAML 文件 | ./blocks/block_catalog.yaml |
| 默认图块 | 当回路无法匹配任何规则时使用的图块名 | LOOP_PWR_3P_01 |

#### 布局参数

| 参数 | 说明 | 默认值 | 单位 |
|------|------|--------|------|
| 图纸幅面 | 选择 A2/A1/A0，"自动选择"表示根据回路数量自动决定 | 自动选择 | - |
| 母线 X | 母线（垂直粗线）在图纸上的 X 坐标 | 100 | mm |
| 图块偏移 X | 图块距母线右侧的偏移量 | 15 | mm |
| 回路间距 | 相邻两个回路图块之间的垂直距离 | 70 | mm |
| 启用分组分隔标签 | 在不同负荷类型的分组之间插入文字标签 | 勾选 | - |

#### 图框/标题栏

| 选项 | 说明 |
|------|------|
| 插入图框 | 是否在图纸左下角插入图框图块 |
| 项目名称 | 写入图框的 PROJECT_NAME 属性 |
| 设计人 | 写入图框的 DESIGNER 属性 |

> 日期自动填写为当天日期。

#### 工具栏按钮

## 5. 命令行 CLI 使用详解

### 5.1 基本语法

```
python main.py <Excel文件路径> [选项]
```

### 5.2 参数说明

| 参数 | 必填 | 说明 |
|------|------|------|
| `<Excel文件路径>` | 是 | 回路清单 .xlsx 文件路径 |

### 5.3 选项说明

| 选项 | 缩写 | 默认值 | 说明 |
|------|------|--------|------|
| `--config PATH` | `-c` | ./config.yaml | 指定配置文件路径 |
| `--output PATH` | `-o` | 配置文件中的设置 | 指定输出 DWG 路径（覆盖配置文件） |
| `--dry-run` | - | 关闭 | 仅校验预览，不连接 AutoCAD |
| `--log-level LEVEL` | - | INFO | 日志级别: DEBUG / INFO / WARNING / ERROR |
| `--gui` | - | 关闭 | 启动图形界面模式 |
| `--version` | - | - | 显示版本号 |
| `--help` | `-h` | - | 显示帮助信息 |

### 5.4 使用示例

```bash
# 基本用法（使用默认配置）
python main.py "D:\项目\配电柜回路表.xlsx"

# 指定配置文件和输出路径
python main.py "回路表.xlsx" -c "my_config.yaml" -o "D:\输出\系统图.dwg"

# 仅校验，不生成 DWG
python main.py "回路表.xlsx" --dry-run

# 调试模式（显示详细日志）
python main.py "回路表.xlsx" --dry-run --log-level DEBUG

# 启动图形界面
python main.py --gui
# 或直接
python gui.py
```

### 5.5 执行流程

```
1. 加载配置文件 (config.yaml)
2. 读取 Excel 回路清单 → 列名匹配 → 逐行校验
3. 加载图块目录 (block_catalog.yaml)
4. 按 负荷类型+极数 匹配图块
5. 构建每个回路的图块属性
6. 计算布局坐标（分组排序、幅面选择）
7. 连接 AutoCAD → 导入图块库 → 插入图块 → 绘制母线 → 插入图框 → 保存 DWG
8. 生成 HTML 报告
```

## 6. Excel 回路清单填写规范

### 6.1 基本要求

| 项目 | 要求 |
|------|------|
| 文件格式 | .xlsx（不支持 .xls） |
| Sheet 名称 | "回路清单"（可在配置文件中修改） |
| 第 1 行 | 列标题 |
| 第 2 行起 | 回路数据 |
| 合并单元格 | 不允许在数据区域使用 |

### 6.2 标准列定义

| 列 | 标准列名 | 可用别名 | 数据类型 | 必填 |
|----|----------|----------|----------|------|
| A | 回路编号 | 编号, ID, 回路ID | 文本 | 是 |
| B | 回路名称 | 名称, 用途, 负荷名称, 用电设备 | 文本 | 是 |
| C | 负荷类型 | 类型, 负荷类别 | 枚举 | 是 |
| D | 额定功率 | 功率(kW), Pe, 设备功率 | 数值 | 是 |
| E | 额定电流 | 电流(A), Ijs, 计算电流 | 数值 | 是 |
| F | 断路器型号 | 开关型号, 断路器 | 文本 | 是 |
| G | 断路器极数 | 极数, P数, Poles | 整数 | 是 |
| H | 脱扣器电流 | 脱扣器(A), Ir, 整定电流 | 数值 | 是 |
| I | CT变比 | 互感器变比, CT, 电流互感器 | 文本 | 是 |
| J | 电缆型号 | 电缆, 线缆型号 | 文本 | 是 |
| K | 电缆规格 | 截面, 规格, 截面积 | 文本 | 是 |
| L | 变频器型号 | 变频器, VFD型号 | 文本 | 否 |
| M | 变频器功率 | VFD功率(kW) | 数值 | 否 |
| N | 备注 | Remark, 说明 | 文本 | 否 |

> **列顺序不强制**：程序通过列名匹配，不依赖列的位置。

> **列名模糊匹配**：程序使用三级匹配策略——精确匹配 → 别名匹配 → 包含匹配。例如列标题写"开关型号"也能正确识别为"断路器型号"。

### 6.3 负荷类型枚举值

| 值 | 含义 | 是否需要变频器 |
|----|------|----------------|
| 动力 | 电动机/动力配电回路 | 否 |
| 照明 | 照明配电回路 | 否 |
| 变频 | 含变频器驱动回路 | **是** |
| 空调 | 空调/暖通回路 | 否 |
| 插座 | 插座配电回路 | 否 |
| 备用 | 预留备用回路 | 否 |
| 电容补偿 | 无功补偿回路 | 否 |

### 6.4 数据校验规则

| 校验项 | 规则 | 违反后果 |
|--------|------|----------|
| 回路编号唯一性 | 不得重复 | 后续相同编号的行被跳过 |
| 必填字段非空 | 所有必填列不可为空 | 该行被跳过 |
| 负荷类型合法 | 必须在枚举表内 | 该行被跳过 |
| 变频器一致性 | 负荷类型=变频时，变频器型号必填 | 该行被跳过 |
| 数值范围 | 功率>0，电流>0，极数=1~4 | 该行被跳过 |
| 列名匹配率 | 至少 70% 必填列可被识别 | 程序退出并报错 |

### 6.6 转置格式 Excel

v1.1 新增对**转置格式** Excel 的支持。部分设计院导出的 Excel 采用"参数为行、回路为列"的布局，与标准格式正好转置。

#### 识别方式

程序通过 A1 单元格内容自动判断格式：

| A1 内容 | 判定格式 |
|---------|----------|
| 包含"参数"二字 | 转置格式 |
| 其他内容 | 标准格式 |

#### 转置格式布局

| | A 列 | B 列 | C 列 | D 列 | ... |
|---|------|------|------|------|-----|
| 第 1 行 | 参数 | 回路1 | 回路2 | 回路3 | ... |
| 第 2 行 | 回路编号 | L1 | L2 | L3 | ... |
| 第 3 行 | 回路名称 | 1#水泵 | 2#水泵 | 1#风机 | ... |
| 第 4 行 | 负荷类型 | 动力 | 动力 | 动力 | ... |
| ... | ... | ... | ... | ... | ... |

以下 A 列参数名会被自动识别（可在 config.yaml 的 `transposed_column_aliases` 中自定义）：

#### 特殊处理

- **负荷类型推断**：如果 Excel 中没有直接的"负荷类型"列，程序会从"运行方式"和"配电形式"字段推断（如"变频运行"→ 变频，"照明"→ 照明）。
- **极数推断**：如果未提供极数，程序从电缆规格推断（如 "4x25+1x16" → 3P）或从电流大小推断。
- **电缆拆分**：如果"电缆"列包含型号和规格（如 "YJV-0.6/1kV 4x185+1x95"），程序自动拆分为电缆型号和电缆规格。
- **CT变比补全**：如果 CT 列仅填写了数字（如 "400"），程序自动补全为 "400/5A"。
- **默认断路器型号**：如果 Excel 中没有断路器型号列，使用配置文件中的 `default_breaker_model`。

## 7. 配置文件 config.yaml 详解

### 7.1 配置文件位置

程序启动时默认读取当前目录下的 `config.yaml`。可通过 `-c` 参数指定其他路径。

### 7.2 完整配置说明

#### autocad — AutoCAD 连接

```yaml
autocad:
  progids:
    - "AutoCAD.Application.24.1"   # AutoCAD 2022
    - "AutoCAD.Application.23.1"   # AutoCAD 2020
  visible: true                     # 连接后 AutoCAD 窗口是否可见
```

程序按列表顺序逐个尝试连接。如果你只用 AutoCAD 2020，可以只保留 23.1。

#### block_library — 图块库

```yaml
block_library:
  path: "./blocks/block_library.dwg"       # 图块库 DWG 文件路径
  catalog: "./blocks/block_catalog.yaml"   # 图块目录 YAML 文件
  title_block_path: "./blocks/title_block.dwg"  # 图框 DWG
  default_block: "LOOP_PWR_3P_01"          # 无匹配时的默认图块
```

#### excel — Excel 读取

```yaml
excel:
  sheet_name: "低压配电系统"       # Sheet 名称（"低压配电系统"与"回路清单"互为备选）
  format_auto_detect: true         # 自动检测标准/转置格式
  header_row: 1               # 列标题行号（标准格式）
  data_start_row: 2           # 数据起始行号（标准格式）
  min_match_ratio: 0.7        # 必填列最低匹配率（70%）
  default_breaker_model: "NSX100N"  # Excel 中无断路器型号时的默认值
  column_aliases:             # 列名别名映射（标准格式）
    回路编号: circuit_id
    编号: circuit_id
    ...
  transposed_column_aliases:  # 转置格式 A 列参数名映射
    回路编号: circuit_id
    回路名称: circuit_name
    运行方式: load_type
    额定功率: rated_power_kw
    设备容量(kW): rated_power_kw
    计算电流(A): rated_current_a
    电缆: cable_full
    ...
```

#### block_mapping — 图块映射规则

```yaml
block_mapping:
  default_block: "LOOP_PWR_3P_01"
  rules:
    - match: {load_type: "动力", poles: 3}
      block: "LOOP_PWR_3P_01"
    - match: {load_type: "变频", poles: 3}
      block: "LOOP_VFD_3P_01"
    # ... 更多规则
```

匹配规则：按列表顺序（优先级从高到低）逐条匹配。对每条回路取 `load_type`（负荷类型）和 `poles`（极数）与规则的 `match` 条件比较，首个命中的规则决定使用哪个图块。

全部规则都不匹配时，使用 `default_block` 并在日志和报告中记录警告。

#### layout — 布局

```yaml
layout:

  vertical_spacing: 70        # 回路垂直间距 (mm)
  margins: {left: 25, right: 25, top: 15, bottom: 30}
```

#### sort — 排序

```yaml
sort:
  group_by: "load_type"
  group_order: ["动力", "变频", "空调", "照明", "插座", "备用", "电容补偿"]

`{type}` 会被替换为实际的负荷类型名称，如"动力 配电"。

```

## 8. 图块库创建规范

### 8.1 图块粒度

**核心设计**：一个图块 = 一个完整配电回路。

每个回路图块包含以下元素：

- 母排接入点（图块顶部，作为插入基准点）
- 电流互感器 (CT)
- 断路器 (QF)
- 变频器 (VFD) — 仅变频回路图块
- 电缆出线端
- 负荷标注区域

### 8.2 图块命名规范

格式：`LOOP_<负荷类型缩写>_<极数>_<序号>`

| 负荷类型 | 缩写 | 示例 |
|----------|------|------|
| 动力 | PWR | LOOP_PWR_3P_01 |
| 照明 | LGT | LOOP_LGT_1P_01 |
| 变频 | VFD | LOOP_VFD_3P_01 |
| 空调 | AC | LOOP_AC_3P_01 |
| 插座 | SKT | LOOP_SKT_1P_01 |
| 备用 | SPR | LOOP_SPR_3P_01 |
| 电容补偿 | CAP | LOOP_CAP_3P_01 |

### 8.3 图块属性 Tag

每个图块必须包含以下属性（Attribute）：

| 属性 Tag | 说明 | 示例值 |
|----------|------|--------|
| CIRCUIT_ID | 回路编号 | L1 |
| CIRCUIT_NAME | 回路名称 | 1#水泵 |
| BREAKER_MODEL | 断路器型号 | NSX100N |
| BREAKER_POLES | 极数 | 3P |
| BREAKER_Ir | 脱扣整定电流 | 32.0A |
| CABLE_TYPE | 电缆型号 | YJV |
| CABLE_SECTION | 电缆规格 | 4x25+1x16 |
| LOAD_POWER | 负荷功率 | 15.00kW |
| LOAD_CURRENT | 计算电流 | 28.50A |
| LOAD_TYPE | 负荷类型 | 动力 |
| CT_RATIO | CT变比 | 50/5A |

变频回路额外属性：

| 属性 Tag | 说明 | 示例值 |
|----------|------|--------|
| VFD_MODEL | 变频器型号 | ATV320 |
| VFD_POWER | 变频器功率 | 30.00kW |

v1.1 新增可选属性（根据 Excel 数据自动填写，为空时不写入）：

| 属性 Tag | 说明 | 示例值 |
|----------|------|--------|
| CABINET_CODE | 配电柜编号 | AP-1 |
| DISTRIBUTION_TYPE | 配电形式 | 放射式 |
| OPERATION_MODE | 运行方式 | 直接启动 |
| BREAKER_FRAME | 断路器壳架电流 | 100A |
| CONTACTOR | 接触器型号 | LC1D32 |
| THERMAL_RELAY | 热继电器型号 | LRD32 |
| POWER_MONITORING | 电力监测 | 是 |
| CABLE_NUMBER | 电缆编号 | DL-001 |

### 8.4 图块插入点

所有回路图块的**插入基准点**统一为**母排接入点**（图块顶部母线连接处）。这样沿垂直母线等距排列时，各回路自然对齐。

### 8.5 图块目录文件

`blocks/block_catalog.yaml` 描述每个图块的元信息。程序据此验证图块是否存在、确定该图块需要填写哪些属性。

可通过 GUI 菜单 **工具 > 生成示例图块目录** 自动生成模板文件。

### 8.6 图框图块

图框图块命名 `TITLE_BLOCK`，包含以下属性：

| 属性 Tag | 说明 |
|----------|------|
| PROJECT_NAME | 项目名称 |
| DRAWING_NO | 图号 |
| DATE | 日期 |
| DESIGNER | 设计人 |

## 9. 布局算法与图纸输出说明

### 9.1 坐标系

- 原点 (0,0) = 图纸左下角
- X 轴向右，Y 轴向上
- 母线为垂直线，位于图纸左侧

### 9.2 排列规则

1. 按负荷类型分组，顺序：动力 → 变频 → 空调 → 照明 → 插座 → 备用 → 电容补偿
2. 组内按回路编号自然排序（L1, L2, L10 而非 L1, L10, L2）
3. 可选：分组之间插入分隔标签文字

### 9.3 自动幅面选择

程序根据回路数量自动计算所需图纸高度，选择能容纳的最小标准幅面：

| 幅面 | 尺寸 (mm) | 大约容纳回路数 |
|------|-----------|----------------|
| A2 | 594 x 420 | ~5 个 |
| A1 | 841 x 594 | ~8 个 |
| A0 | 1189 x 841 | ~11 个 |

> 具体数量受间距和分组标签影响。也可在 GUI 或配置文件中手动指定幅面。

### 9.4 母线绘制

- 位置：X = bus_x（默认 100mm）
- 范围：从最上方回路到最下方回路
- 线宽：0.5mm 粗实线（AutoCAD Lineweight = 50）

### 9.5 输出文件

| 文件 | 说明 |
|------|------|
| 配电系统图.dwg | AutoCAD 图纸文件 |
| report.html | HTML 处理报告 |
| app.log | 运行日志 |

## 10. 错误处理与容错策略

### 10.1 核心原则

**环境级问题直接退出，单回路问题跳过继续。**

### 10.2 错误分类

#### 退出级错误（程序终止）

| 错误 | 处理方式 |
|------|----------|
| Excel 文件不存在 | 打印错误 → 退出 |
| Sheet 未找到 | 列出可用 Sheet → 退出 |
| 必填列匹配率不足 70% | 列出已识别列 → 退出 |
| 全部回路校验失败 | 打印汇总 → 退出 |
| 图块库 DWG 不存在 | 打印错误 → 退出 |
| AutoCAD 未运行 | 提示启动 → 退出 |
| AutoCAD 版本不兼容 | 列出支持版本 → 退出 |

#### 跳过级错误（继续处理）

| 错误 | 处理方式 |
|------|----------|
| 单个回路校验失败 | 跳过该行，记录错误 |
| 图块映射无匹配 | 使用默认图块，记录警告 |
| 单个图块插入失败 | 跳过该回路，继续 |
| 属性写入失败 | 记录警告，继续 |

### 10.3 HTML 报告

每次运行后自动生成 `output/report.html`，包含：

- 处理时间戳与源文件
- 汇总统计（总数/成功/警告/跳过）
- 图块使用统计
- 成功回路清单（编号、名称、图块名、坐标）
- 错误/跳过多细（行号、错误类型、错误消息）
- 警告明细

### 10.4 日志文件

运行日志保存在 `logs/app.log`，格式：

## 11. 常见问题排查（FAQ）

### Q1: "AutoCAD 连接失败"怎么办？

1. 确认 AutoCAD 已启动
2. 检查 config.yaml 中的 progids 是否匹配你的 AutoCAD 版本
3. 尝试以管理员权限运行 PDSG
4. 确认 pyautocad 和 comtypes 已正确安装

### Q2: "Sheet 未找到"怎么办？

1. 打开 Excel 确认 Sheet 名称
2. 在 config.yaml 中修改 `excel.sheet_name` 为实际名称
3. 或在 GUI 配置中修改

### Q3: "必填列匹配率不足"怎么办？

1. 检查 Excel 列标题是否与标准列名一致
2. 在 config.yaml 的 `column_aliases` 中添加你的列名映射
3. 例如你的列标题是"开关"，添加：`开关: breaker_model`

### Q4: 某些回路被跳过了怎么办？

1. 查看 HTML 报告中的"错误/跳过多细"表
2. 根据错误类型修正 Excel 数据
3. 常见问题：空值、非法负荷类型、重复编号、变频器回路缺少变频器型号

### Q5: 生成的图纸回路重叠或间距不合理？

1. 在 GUI 配置中调整"回路间距"参数（默认 70mm）
2. 如果回路很多，考虑手动指定更大的幅面（如 A0）
3. 减小母线 X 坐标可以给回路更多水平空间

### Q6: 如何添加新的图块类型？

1. 在 AutoCAD 中创建新图块，按命名规范和属性规范设置
2. 在 `block_catalog.yaml` 中添加图块定义
3. 在 `config.yaml` 的 `block_mapping.rules` 中添加匹配规则
4. 将新图块保存到 `block_library.dwg` 中

### Q7: Dry-Run 和正式生成有什么区别？

| 对比 | Dry-Run | 正式生成 |
|------|---------|----------|
| 读取 Excel | 是 | 是 |
| 图块映射 | 是 | 是 |
| 布局计算 | 是 | 是 |
| 连接 AutoCAD | **否** | 是 |
| 生成 DWG | **否** | 是 |
| 生成报告 | 是 | 是 |
| 需要 AutoCAD 运行 | **否** | 是 |

建议先用 Dry-Run 确认数据和映射无误，再执行正式生成。

### Q8: 我的 Excel 是"参数在行、回路在列"的格式，怎么办？

v1.1 已支持转置格式的自动检测。只要勾选"自动检测 Excel 格式"（默认已勾选），程序会自动识别 A1 单元格含"参数"字样的 Excel 为转置格式，并正确解析。

如果自动检测有误，可以在 config.yaml 中设置 `format_auto_detect: false` 并手动调整。

### Q9: Sheet 名称是"低压配电系统"但程序说找不到？

"低压配电系统"和"回路清单"互为自动备选。如果你的 Sheet 名称不是这两个，请在 GUI 配置标签页的"Sheet 名称"中输入实际名称，或在 config.yaml 中修改 `excel.sheet_name`。

### 13.1 AutoCAD 启动与模板设置

#### 启动 AutoCAD

1. 双击桌面 AutoCAD 快捷方式启动程序
2. 在起始页选择 **"新建"** → 选择模板 `acad.dwt`（公制）或 `acadiso.dwt`（ISO 公制）
3. 如果使用 AutoCAD 2022，界面默认使用深色主题；2020 类似

#### 单位设置

在命令行输入以下命令确认单位为毫米：

```
UNITS
```

在弹出的对话框中：
- 长度类型：小数
- 长度精度：0
- 插入比例：毫米

#### 保存为工作模板

设置好单位和图层后，另存为模板文件 `PDSG_Template.dwt`，以后每次新建图纸时可直接使用。

### 13.2 图层与线型规划

合理的图层规划是专业出图的基础。建议按以下方案设置图层：

#### 推荐图层方案

| 图层名 | 颜色 | 线型 | 线宽 | 用途 |
|--------|------|------|------|------|
| BUSBAR | 白色(7) | Continuous | 0.50mm | 母线（水平/垂直粗线） |
| CIRCUIT | 青色(4) | Continuous | 0.25mm | 回路图块轮廓 |
| BREAKER | 黄色(2) | Continuous | 0.25mm | 断路器符号 |
| CT | 绿色(3) | Continuous | 0.18mm | 电流互感器符号 |
| CABLE | 白色(7) | Continuous | 0.18mm | 电缆连线 |
| TEXT | 洋红(6) | Continuous | 0.13mm | 文字标注 |
| ATTRIBUTE | 红色(1) | Continuous | 0.13mm | 图块属性文字 |
| TITLEBLOCK | 白色(7) | Continuous | 0.35mm | 图框 |
| DIMENSION | 蓝色(5) | Continuous | 0.13mm | 尺寸标注（预留） |
| DEFPOINTS | 白色(7) | Continuous | 默认 | 尺寸标注基准点（AutoCAD 自动创建） |

#### 创建图层步骤

1. 命令行输入 `LAYER`（或点击"图层特性管理器"按钮）
2. 点击"新建图层"按钮（太阳图标），依次输入上述图层名
3. 逐个设置颜色：点击颜色单元格 → 选择对应索引颜色
4. 逐个设置线宽：点击线宽单元格 → 选择对应线宽
5. 将 CIRCUIT 图层设为当前图层（选中后点击"置为当前"按钮）
6. 点击"确定"关闭

#### 加载线型（如需要虚线）

如果需要使用虚线（如备用回路表示），在命令行输入：

```
LINETYPE
```

点击"加载"，选择 `ACAD_ISO02W100`（虚线）和 `ACAD_ISO04W100`（点划线），确认后关闭。

### 13.3 三相动力回路图块创建教程

本节以一个标准的三相动力回路（LOOP_PWR_3P_01）为例，完整演示图块创建过程。

#### 第一步：绘制回路图形

切换到 **CIRCUIT** 图层，按以下坐标绘制各元件（以插入点为原点 0,0）：

**1. 母线接入点（顶部）**

图块的插入基准点，也是与垂直母线的连接点。

```
命令: POINT
指定点: 0,0
```

**2. 水平引入线（从母线到断路器）**

```
命令: LINE
指定第一点: 0,0
指定下一点: 30,0
```

**3. 电流互感器 CT 符号**

CT 用两个同心圆表示，圆心在引入线上：

```
命令: CIRCLE
指定圆心: 15,0
指定半径: 3

命令: CIRCLE
指定圆心: 15,0
指定半径: 5
```

**4. 断路器 QF 符号**

断路器用矩形加斜线表示：

```
命令: RECTANG
指定第一个角点: 25,-6
指定另一个角点: 35,6

命令: LINE
指定第一点: 27,-4
指定下一点: 33,4
```

**5. 垂直下行线（断路器到电缆出线）**

```
命令: LINE
指定第一点: 30,0
指定下一点: 30,-30
```

**6. 电缆出线端标记**

```
命令: CIRCLE
指定圆心: 30,-30
指定半径: 2
```

**7. 负荷标注区域（右侧水平线）**

```
命令: LINE
指定第一点: 30,-25
指定下一点: 50,-25
```

> **提示**：以上坐标仅为参考。实际绘制时可根据图块大小和美观需要调整。关键是保持插入点（0,0）在图块顶部，以便沿母线排列时自然对齐。

#### 第二步：添加属性定义

切换到 **ATTRIBUTE** 图层，使用 `ATTDEF`（属性定义）命令为图块添加可变文字字段。

**命令调用**：命令行输入 `ATTDEF`，弹出"属性定义"对话框。

**必须定义的 11 个属性**：

| 序号 | 标记(Tag) | 提示(Prompt) | 默认值 | 文字高度 | 对齐方式 |
|------|-----------|-------------|--------|----------|----------|
| 1 | CIRCUIT_ID | 回路编号 | L1 | 3.0 | 左对齐 |
| 2 | CIRCUIT_NAME | 回路名称 | 设备名 | 3.0 | 左对齐 |
| 3 | BREAKER_MODEL | 断路器型号 | NSX100N | 2.5 | 左对齐 |
| 4 | BREAKER_POLES | 极数 | 3P | 2.5 | 左对齐 |
| 5 | BREAKER_Ir | 脱扣整定电流 | 32.0A | 2.5 | 左对齐 |
| 6 | CABLE_TYPE | 电缆型号 | YJV | 2.5 | 左对齐 |
| 7 | CABLE_SECTION | 电缆规格 | 4x25+1x16 | 2.5 | 左对齐 |
| 8 | LOAD_POWER | 负荷功率 | 15.00kW | 2.5 | 左对齐 |
| 9 | LOAD_CURRENT | 计算电流 | 28.50A | 2.5 | 左对齐 |
| 10 | LOAD_TYPE | 负荷类型 | 动力 | 2.5 | 左对齐 |
| 11 | CT_RATIO | CT变比 | 50/5A | 2.5 | 左对齐 |

**逐个添加步骤**：

1. 输入 `ATTDEF`，在对话框中填写标记、提示、默认值
2. 文字设置中选择"左对齐"，高度设为上表值
3. 点击"确定"后，在图形的合适位置点击放置属性
4. 建议放置位置：

```
CIRCUIT_ID     → (40, -5)   断路器右侧上方
CIRCUIT_NAME   → (40, -9)   回路名称
BREAKER_MODEL  → (40, -13)  断路器型号
BREAKER_POLES  → (40, -16)  极数
BREAKER_Ir     → (40, -19)  脱扣电流
CABLE_TYPE     → (40, -22)  电缆型号
CABLE_SECTION  → (40, -25)  电缆规格（与负荷标注线对齐）
LOAD_POWER     → (52, -22)  功率
LOAD_CURRENT   → (52, -25)  电流
LOAD_TYPE      → (52, -19)  负荷类型
CT_RATIO       → (5, -6)    CT 变比（靠近 CT 符号）
```

> **重要**：Tag 名称必须与上表完全一致（区分大小写），PDSG 通过这些 Tag 名匹配并写入属性值。

#### 第三步：创建图块

1. 确保所有图形元素和属性定义都已完成
2. 命令行输入 `BLOCK`（或 `B`），弹出"块定义"对话框
3. 填写参数：
   - **名称**：`LOOP_PWR_3P_01`
   - **基点**：点击"拾取点"按钮 → 点击图形中的 (0,0) 点（母排接入点）
   - **对象**：点击"选择对象"按钮 → 框选所有图形和属性
   - **行为**：选择"注释性"（可选，用于不同比例出图）
   - **块单位**：毫米
4. 点击"确定"

> **注意**：创建图块后，原图形会变为图块引用。如果需要修改，使用 `BEDIT`（块编辑器）命令双击图块进入编辑模式。

### 13.4 属性定义（ATTDEF）参数详解

#### 属性模式说明

在"属性定义"对话框中，有几个重要的模式选项：

| 模式 | 说明 | 推荐设置 |
|------|------|----------|
| 不可见 | 属性值在图纸上不显示，但仍存储在图块中 | 仅对纯数据属性勾选 |
| 固定 | 属性值不可修改（创建后固定） | 不勾选（PDSG 需要写入） |
| 验证 | 插入时要求二次确认 | 不勾选 |
| 预设 | 插入时不提示，直接使用默认值 | **勾选**（PDSG 通过程序写入） |
| 锁定位置 | 属性位置不可拖动 | 勾选 |
| 多行文字 | 允许属性为多行文字 | 不勾选 |

#### 文字样式

建议创建专用文字样式：

```
命令: STYLE
```

- **样式名**：PDSG_TEXT
- **字体名**：`gbcbig.shx`（国标大字体）或 `SimSun`（宋体）
- **宽度因子**：0.8
- **倾斜角度**：0

> 中文标注务必使用支持中文的字体，否则生成的 DWG 中中文会显示为问号。

### 13.5 批量创建其他类型图块

按照 13.3 的方法，逐个创建其余图块。下表列出每种图块的差异点：

#### 图块清单及差异

| 图块名 | 与标准动力回路的差异 | 额外属性 |
|--------|---------------------|----------|
| LOOP_PWR_3P_01 | 标准三相动力（教程示例） | — |
| LOOP_PWR_4P_01 | 四极断路器，断路器矩形加宽 | — |
| LOOP_VFD_3P_01 | 在断路器与电缆之间增加变频器符号（矩形内写 VFD） | VFD_MODEL, VFD_POWER |
| LOOP_LGT_1P_01 | 单相，CT 和断路器缩小，断路器为 1P | — |
| LOOP_LGT_3P_01 | 三相照明，外形与动力相似 | — |
| LOOP_AC_3P_01 | 三相空调，外形与动力相似 | — |
| LOOP_CAP_3P_01 | 电容补偿，可在电缆端增加电容符号（两条平行线） | — |

#### 变频器图块的特殊绘制

变频器回路在断路器与电缆之间增加一个变频器符号：

```
命令: RECTANG
指定第一个角点: 28,-12
指定另一个角点: 38,-22

命令: TEXT
指定起点: 30,-15
指定高度: 3
输入文字: VFD
```

变频器的额外属性定义：

| 标记(Tag) | 提示 | 默认值 | 放置位置 |
|-----------|------|--------|----------|
| VFD_MODEL | 变频器型号 | ATV320 | (42, -15) |
| VFD_POWER | 变频器功率 | 30.00kW | (42, -18) |

#### 单相图块的简化

单相图块（1P）与三相的主要区别：

- CT 只用一个（而非三个），圆心仍在引入线上
- 断路器矩形宽度缩小（表示单极）
- 属性 BREAKER_POLES 默认为 "1P"

### 13.6 图块库 DWG 文件组织

所有图块创建完成后，需要将它们统一保存到一个 DWG 文件中，作为 PDSG 的图块库。

#### 组织方式

1. 新建一个空白 DWG 文件（使用之前保存的模板）
2. 使用 `INSERT` 命令或 `DesignCenter`（Ctrl+2）将各图块逐个插入到此文件中
3. 每个图块插入到不同位置，保持间距（如 X 方向间隔 100mm）
4. 插入后删除图块引用（图块定义仍保留在文件中）
5. 保存为 `blocks/block_library.dwg`

> **技巧**：也可以使用 `WBLOCK` 命令将每个图块单独导出为 DWG，然后在图块库文件中用 `INSERT` 导入。

#### 验证图块库

在命令行输入以下命令确认图块已存在于文件中：

```
INSERT
```

在弹出的对话框中，点击下拉列表，应能看到所有已定义的图块名称。

#### 图块目录 YAML 文件

在 `blocks/block_catalog.yaml` 中列出所有图块的元信息。可执行 GUI 菜单 **"工具 > 生成示例图块目录"** 自动生成模板，然后按需修改。

### 13.7 图框图块创建

图框用于包围整张图纸，包含项目信息。

#### 创建步骤

1. 切换到 **TITLEBLOCK** 图层
2. 绘制外框矩形（以 A2 为例）：

```
命令: RECTANG
指定第一个角点: 0,0
指定另一个角点: 594,420
```

3. 绘制内框（左边距 25mm，其余 10mm）：

```
命令: RECTANG
指定第一个角点: 25,10
指定另一个角点: 584,410
```

4. 绘制标题栏（右下角）：

```
命令: RECTANG
指定第一个角点: 384,10
指定另一个角点: 584,80
```

5. 在标题栏内添加分隔线和固定文字（项目名称、图号、日期、设计人等）

6. 为可变信息添加属性定义：

| 标记(Tag) | 提示 | 默认值 | 放置位置 |
|-----------|------|--------|----------|
| PROJECT_NAME | 项目名称 | — | (484, 60) |
| DRAWING_NO | 图号 | — | (484, 45) |
| DATE | 日期 | — | (484, 30) |
| DESIGNER | 设计人 | — | (484, 15) |

7. 选中所有图形和属性，输入 `BLOCK`，名称填 `TITLE_BLOCK`，基点设为 (0,0)

8. 保存为 `blocks/title_block.dwg`

### 13.8 PDSG 连接 AutoCAD 配置

#### AutoCAD 2022 配置

AutoCAD 2022 的 ProgID 为 `AutoCAD.Application.24.1`。

**连接步骤**：

1. 启动 AutoCAD 2022
2. 打开图块库文件 `blocks/block_library.dwg`（PDSG 需要从中复制图块）
3. 启动 PDSG（GUI 或 CLI）
4. PDSG 会自动查找并连接 AutoCAD

**ProgID 验证**：在 AutoCAD 命令行输入以下代码可查看 ProgID：

```
(vlax-product-key)
```

返回值中应包含 "24.1"。

#### AutoCAD 2020 配置

AutoCAD 2020 的 ProgID 为 `AutoCAD.Application.23.1`。

如果你只安装了 AutoCAD 2020，修改 config.yaml：

```yaml
autocad:
  progids:
    - "AutoCAD.Application.23.1"   # 仅 2020
  visible: true
```

#### 双版本共存

如果同时安装了 2020 和 2022，config.yaml 中的默认配置即可工作：

```yaml
autocad:
  progids:
    - "AutoCAD.Application.24.1"   # 优先尝试 2022
    - "AutoCAD.Application.23.1"   # 2022 不存在则尝试 2020
```

#### COM 接口权限

如果遇到"COM 连接失败"，尝试以下操作：

1. 以**管理员身份**运行 PDSG
2. 在 AutoCAD 中确认允许外部程序连接：
   - 命令行输入 `SECURELOAD`，设为 `0`
   - 命令行输入 `LISPINIT`，设为 `0`（如果需要加载 LISP）
3. 关闭 Windows 防火墙或添加例外

### 13.9 常见问题与排查

#### Q: 创建的图块插入后属性不显示？

A: 检查 `ATTMODE` 系统变量，设为 `1`（正常显示）。如果设为 `0`，所有属性都不可见。

```
ATTMODE
输入新值: 1
```

#### Q: 图块中的中文显示为问号？

A: 文字样式使用了不支持中文的字体。修改文字样式为 `SimSun`（宋体）或 `gbcbig.shx` + `bigfont`。

```
STYLE
选择样式: PDSG_TEXT
字体名: SimSun
```

#### Q: PDSG 报"图块 XXX 在库中不存在"？

A: 
1. 确认图块库 DWG 中的图块名称与 config.yaml 中的映射规则完全一致
2. 在 AutoCAD 中输入 `INSERT`，检查下拉列表中是否有该图块
3. 图块名称区分大小写，确保 `LOOP_PWR_3P_01` 不是 `Loop_Pwr_3P_01`

#### Q: AutoCAD 2022 连接超时？

A: AutoCAD 2022 首次启动较慢（需要加载许可证服务），等待完全加载后再运行 PDSG。如果仍超时，可以先手动启动 AutoCAD 并等待界面完全加载后，再运行 PDSG。

#### Q: 生成的图纸中图块重叠？

A: 
1. 检查回路间距设置（默认 70mm），如果图块高度超过此值需要加大间距
2. 在 config.yaml 中调整：

```yaml
layout:
  vertical_spacing: 90    # 加大间距
```

#### Q: 如何验证 PDSG 能正确连接 AutoCAD？

A: 在 PDSG GUI 中执行"校验预览 (Dry-Run)"。Dry-Run 不连接 AutoCAD，但可以验证 Excel 读取、图块映射、布局计算是否正确。确认 Dry-Run 无误后，再执行"生成 DWG"进行正式连接。
