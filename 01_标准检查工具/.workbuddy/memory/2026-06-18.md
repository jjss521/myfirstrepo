# 2026-06-18 工作记录

## 新增功能：直接文本输入标准

**需求**：用户可以直接粘贴标准编号列表，无需截图OCR，跳过OCR直接进入网站查询+报告生成。

### 修改文件

1. **standard_parser.py** — 新增 `parse_standards_from_text(text, source)` 函数
   - 从纯文本中提取标准编号和名称
   - 支持每行一条、逗号分隔多条等灵活格式
   - 自动去重，置信度固定为 1.0
   - 兼容书名号、冒号等分隔符自动清洗

2. **main.py** — 新增 CLI 参数 `--text` 和 `--text-file`
   - `--text "GB 50016-2014 建筑设计防火规范"` 直接输入
   - `--text-file standards.txt` 从文件读取
   - 文本输入模式下直接跳过OCR，进入网站查询阶段
   - 支持 `\\n` 转义换行

3. **gui_app.py** — 新增输入模式切换
   - 顶部增加「📷 截图模式」/「📝 文本输入」切换按钮
   - 文本模式下显示大文本框，隐藏截图目录/OCR置信度
   - `_switch_mode()` 控制UI切换
   - `_run_pipeline()` 支持 text_input 参数，文本模式下跳过OCR

### 测试结果
- 3个文件全部通过 `py_compile` 语法检查
- `parse_standards_from_text` 功能测试通过：7行输入 → 6条去重标准，所有编号和名称正确
