# 项目台账 — 独立桌面程序

内网门户「工时管理 → 项目台账」页面的**离线独立版**。无需登录网页，双击 `项目台账.exe` 即可操作全部功能。

## 文件清单
- `项目台账.exe` — 主程序（双击运行）
- `project_ledger.db` — 本地数据库（首次运行自动生成，含样本数据；与 EXE 同目录）
- `main.py` — 源码（如需自行重新打包）
- `README.md` — 本说明

> 备份/迁移：把 `项目台账.exe` 和 `project_ledger.db` 一起拷贝即可，数据库就是你的全部数据。

|---------|-----------|

| 高级搜索（搜索/重置/存为模板） | ✅ 高级搜索面板 + 存为模板/载入 |

## 第二个标签页：工时填报
复刻门户「工时管理 → 工时填报」表单，作为独立标签页运行，数据同样存入本地 `project_ledger.db`（表 `work_hours`）。

**基础信息区（顶部）**
- 填报人：默认「贾瑟」（只读）
- 部门：默认「四院本部」（只读）
- 所属专业所：默认「电气所」（只读）
- 专业所所长：默认「周畅」（只读）
- 技术职称：下拉（教授级高级工程师 / 高级工程师 / 工程师 / 助理工程师）
- 职称级别：下拉（正高级 / 副高级 / 中级 / 初级）
- 从事专业：下拉（电气及自动化 / 电气 / 自动化 / 给排水 / 结构 / 建筑 / 暖通）
- 工时所属日期：日期选择，**必填**
- 填报日期：默认今天（只读，保存时写入）

**工时信息明细表（可多行）**
- 列：勾选 / 序号 / 项目名称 / 项目编号 / 任务类别 / 工时(h) / 出差工时(d) / 实际出差工时(天) / 工作内容 / 备注
- 「项目名称」下拉取自项目台账（含请假/管理/行政/学习 4 个非项目类预置项），**选中后自动带出项目编号**
- 工具栏：添加行 / 删除行（勾选）/ 复制行
- 按钮：保存 / 保存并新建（清空表单继续填下一天）

**已填报记录**
- 列表展示填报人 / 工时所属日期 / 填报日期 / 明细条数
- 刷新 / 查看明细（弹窗列出每行内容）/ 删除记录

> 详细字段与网页对照见同目录 `工时填报功能详解.md`。

## 使用流程
1. **直接运行**：双击 `项目台账.exe`。首次会自动建库并写入约 40 条样本数据。
2. **载入真实数据**（可选）：在网页端把项目台账导出为 CSV，回到本程序点「导入」选中该 CSV 即可一次性载入全部真实记录（之后完全离线）。
3. **日常操作**：搜索、排序、新增/编辑、批量删除、批量共享、导出。

## 导入 CSV 格式
表头需含以下中文列名（顺序不限，缺列自动留空）：
`项目名称, 项目编号, 工程费用(万元), 设计费(万元), 项目负责人, 项目状态, 是否公共项目, 显示顺序, 项目归属, 项目类别, 创建人`
- 「是否公共项目」填 `是`/`否`
- 工程费用、设计费填数字；显示顺序填整数

## 重新打包（开发者）
```bash
# 在隔离 Python 环境（venv）中
pip install PySide6 PyInstaller
pyinstaller --onefile --windowed --name 项目台账 main.py
# 产物：dist/项目台账.exe
```
