# 低压配电系统 回路参数计算工具

## 当前版本: v6
- 文件: `lowvolt_calc.py`, `dist/低压配电计算工具.exe`
- 技术: Python 3.13 + tkinter 深色主题 + PyInstaller 单文件EXE

## 核心规则
- **Ic = Pe × 2**（简单公式）
- **运行方式**：仅 工频/变频
- **配电形式**：MCC/PC
- **电缆芯数**：3/4/5/3+2/4+1
- **电缆选型**：严格 VBA `SelectCable_Model` 算法（载流量最少根数 + 压降逐级放宽 + startIdx/forceMin2 逻辑）
- **柜号**：基础名-AN01/AN02… 自动编号，72E/柜上限
- **备用回路**：弹窗手动选断路器，无接触器/热继/变频器/CT/监控
- **数据库**：电缆载流量14档 + 低压电器31档，弹窗编辑，默认锁定
- **GUI**：顶部输入面板(水平) + 底部汇总表格(转置17行×N列)
- **柜体合并**：同柜代号+尺寸合并显示

## 用户偏好
